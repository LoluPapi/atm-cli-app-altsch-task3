# ATM CLI APP


This is an ATM CLI App

### To run this program

```bash
go run .
```

### The Welcome page is like this
```
**-**Welcome to the ATM CLI app!**-**
Enter any option to be served!
1. Change Pin
2. Balance
2. Withdraw
3. Deposit
```
